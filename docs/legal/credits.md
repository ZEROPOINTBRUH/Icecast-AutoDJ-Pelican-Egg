## 🛠️ Core Technologies

### Icecast
- **Organization:** Xiph.Org Foundation
- **Website:** https://icecast.org/
- **License:** GNU GPL v2
- **Role:** Streaming media server (core component)

### ezstream
- **Organization:** Xiph.Org Foundation
- **Website:** https://icecast.org/ezstream/
- **License:** GNU GPL v2
- **Role:** Source client for streaming automation

### LAME
- **Project:** LAME MP3 Encoder
- **License:** GNU LGPL
- **Role:** MP3 encoding

### madplay
- **Project:** MAD (MPEG Audio Decoder)
- **License:** GNU GPL
- **Role:** MP3 decoding

### vorbis-tools
- **Organization:** Xiph.Org Foundation
- **License:** BSD-like
- **Role:** OGG Vorbis encoding/decoding

### opus-tools
- **Organization:** Xiph.Org Foundation
- **License:** BSD
- **Role:** OPUS codec support

### FFmpeg
- **Project:** FFmpeg
- **Website:** https://ffmpeg.org/
- **License:** GNU LGPL/GPL
- **Role:** Universal media processing and transcoding
