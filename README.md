## What Does It Do?
Exactly what it says it can do—it plays a consistent radio stream for you, your friends, or even your uncle, for crying out loud. You can shove your music in there, and it just works.

**Enhanced Features**:
- 🎵 Streams your music (still does this, surprisingly)
- 🔐 Actual security (passwords aren't "hackme" anymore!)
- 📊 Configurable everything (14 environment variables!)
- 🚨 Error messages that make sense (revolutionary concept)
- 📚 Documentation (yes, I actually wrote some)
- 🎨 Color-coded installation logs (fancy!)

## How Do I Use It?

### Quick Start (New Improved Version)
1. Import `egg/radio-improved.json` into your Pterodactyl panel
2. Set strong passwords (not "hackme" this time)
3. Upload music to `/music` directory
4. Watch it actually work

**Detailed Instructions**: See `egg/INSTALLATION.md` for the full guide (yes, it exists now)

### Legacy Version
The original egg is still at `egg/radio.json` if you enjoy living dangerously.
