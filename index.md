## 🎵 What is AutoDJ-Extreme?

A professional-grade Icecast radio streaming solution with advanced AutoDJ capabilities, built specifically for Pterodactyl and Pelican Panel. Transform your server into a full-featured radio station with intelligent playlist management, ads rotation, and professional branding.

## 🚀 Quick Start

Get your radio station running in 5 minutes:

1. **Import Egg** - Download `radio-extreme.json` and import to your panel
2. **Create Server** - Allocate port and resources
3. **Configure** - Set passwords and enable features
4. **Upload Music** - Add files to `/music` directory
5. **Start Broadcasting** - Launch and enjoy!
